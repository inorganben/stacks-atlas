## About The Stacks Project

[The Stacks Project](https://stacks.math.columbia.edu/) is a collaborative, LaTeX-written encyclopedic reference for algebraic geometry, aiming to cover everything from commutative algebra and sheaf theory to algebraic stacks and derived categories, with all arguments self-contained and traceable. This project extracts the definitions, lemmas, and theorems together with their dependency relations and organizes them into an interactive 3D star-map.

## Features

- All nodes and dependency edges rendered in a single pass (InstancedMesh + LineSegments)
- Detail panel with KaTeX-rendered original math
- FlexSearch full-text search, tag-based filtering, shortest-path finding (Web Worker)
- Camera presets and automated TOUR
- URL deep-linking, keyboard shortcuts, three quality presets

## Local Deployment

Requires Node.js 18+.

```bash
git clone https://github.com/inorganben/stacks-atlas.git
cd stacks-atlas
npm install
npm run dev      # dev server, default http://localhost:5173
npm run build    # production build into dist/
npm run preview  # preview the production build
```
